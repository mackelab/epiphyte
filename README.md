## What is Epiphyte?

Epiphyte is a domain-specific relational database designed for analyzing spiking and field potential data recorded during movie presentations. 

At its core, Epiphyte is a worked solution for organizing movie annotations and neural data using [DataJoint](https://github.com/datajoint).

We provide an open-source framework for building and remotely deploying a database structured specifically for working with naturalistic and continuous stimuli and neural recordings.
